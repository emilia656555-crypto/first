Основні команди Git

  • git init – створити репозиторій
  • git add . – додати всі файли
  • git commit -m "текст" – зробити коміт
  • git push – відправити на GitHub
  • git pull – завантажити з GitHub
  • git branch – показати гілки
  • git checkout -b ім'я – створити гілку
  • git merge ім'я – злити гілку
