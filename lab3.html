Мета роботи

Зрозуміти основні принципи методу групового урахування аргументів (GMDH) на прикладі простих одновимірних даних та порівняти його ефективність з поліноміальною регресією.

Теоретичні відомості

Метод групового урахування аргументів (GMDH) — це індуктивний алгоритм побудови моделей, який поетапно генерує та відбирає моделі на основі інформаційного критерію.

Інформаційний критерій:
I = σ² / (1 - k²/n)

де:
• σ² — середньоквадратична помилка моделі
• k — кількість параметрів у моделі
• n — розмір вибірки

Методика дослідження

  1. Використано ті самі дані, що й у лабораторній №2: y = sin(x) + 0.5x + шум
  2. Реалізовано алгоритм МГУА для поліномів ступенів 1-8
  3. Для кожної моделі обчислено інформаційний критерій
  4. Обрано модель з мінімальним значенням IC
  5. Порівняно результат з найкращою моделлю з лабораторної №2

Аналіз результатів

  • МГУА успішно вибрав поліном 3-го ступеня як оптимальну модель
  • Інформаційний критерій досягає мінімуму при ступені 3, після чого починає зростати
  • Результати МГУА повністю збігаються з результатами поліноміальної регресії з лабораторної №2
  • IC ефективно враховує як точність моделі, так і її складність

Висновки

  1. Метод МГУА з використанням інформаційного критерію є ефективним інструментом для автоматичного вибору оптимальної моделі
  2. Інформаційний критерій забезпечує баланс між точністю та складністю моделі, запобігаючи перенавчанню
  3. Для одновимірних даних МГУА дав ті самі результати, що й ручний підбір у поліноміальній регресії
  4. Перевагою МГУА є повна автоматизація процесу вибору моделі
